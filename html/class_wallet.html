Detailed Description

A entidade Wallet representa uma carteira de investimentos.

Uma carteira de investimentos é o conjunto de ativos financeiros que um investidor possui, como ações, títulos e fundos. Sua composição depende do perfil do investidor e dos objetivos financeiros.

Para criar a carteira utiliza-se as seguintes informações:

  • Codigo identificador;
  • Nome do usuário;
  • Perfil de investidor.
