Detailed Description

Representa o código de identificação.

Código é um identificador único atribuído a algo com o objetivo de diferenciá-lo dos demais dentro de um sistema. Pode ser composto por números, letras ou uma combinação de ambos, sendo utilizado para facilitar a organização,localização e manipulação de dados. Em contextos financeiros, um código pode representar uma carteira, entre outros.
