Detailed Description

Responsável por armazenar o nome do investidor.

Nome é a designação que identifica uma pessoa, podendo ser composto por um ou mais nomes e sobrenomes. É utilizado para distinguir indivíduos e facilitar a comunicação. No que diz respeito a investimentos, o nome é um dado essencial para identificar o investidor e associá-lo a suas transações financeiras.
