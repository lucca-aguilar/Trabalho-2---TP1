Detailed Description

A entidade Order representa uma ordem de compra ou venda de um ativo financeiro.

Uma ordem é uma instrução dada por um investidor para comprar ou vender um ativo financeiro em uma bolsa de valores. As ordens podem ser de diferentes tipos, como ordens de mercado, ordens limitadas, entre outras.

Para criar a ordem utiliza-se as seguintes informações:

  • Codigo identificador da carteira;
  • Código identificador da Negociação (operação);
  • Data em que a operação foi realizada;
  • Preço do ativo.;
  • Quantidade.
